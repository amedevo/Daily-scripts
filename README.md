# Network Recon Tool v1.0

A professional network reconnaissance and automation script for Kali Linux environments.

## Features
- **Interface Mapping:** Lists all network interfaces and IP addresses using `ip addr`.
- **Wireless Scanning:** Real-time WiFi network discovery via `nmcli`.
- **Modular Design:** Built with Python's `subprocess` for direct system interaction.

## Usage
Ensure you are on a Linux-based system (Kali Linux recommended).

command : python3 network_recon.py


!!!This tool is developed for educational and authorized security testing purposes only. The responsibility for any misuse lies entirely with the user.

# Ağ Keşif Aracı v1.0

Kali Linux ortamları için geliştirilmiş, profesyonel bir ağ keşif ve otomasyon aracıdır.

## Özellikler
- **Arayüz Haritalama:** `ip addr` komutunu kullanarak tüm ağ arayüzlerini ve IP adreslerini listeler.

- **Modüler Yapı:** Sistemle doğrudan etkileşim kurmak için Python'ın `subprocess` kütüphanesi ile optimize edilmiştir.

## Kullanım
